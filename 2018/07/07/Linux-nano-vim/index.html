Linux下简单编辑修改文本文件

发表于 2018-07-07 | 分类于 绝

  Linux下的文本编辑器有很多，此篇文章挑选 nano 和 vim 两个编辑器，简单介绍如何使用这两个编辑器编辑修改文本文件，如项目的配置文件，日志级别的xml文件等。
  在很多 Linux 发行版中，都已经内置了nano 与 vim 命令工具，就不再赘述它们的安装过程了。

  • nano官网
  • vim官网

nano

  nano是一个字符终端的文本编辑器，它比另一个我们要介绍的vim简单很多，在nano的终端界面底部可以看到一些常用的快捷键提示，非常适合Linux初学者使用。
  nano命令可以打开指定文件进行编辑，默认情况下它会自动断行，即在一行中输入过长的内容时会自动拆分成几行。使用这种方式处理某些文件可能会带来问题，所以使用nano打开文本文件时，建议加上 -w 选项。
  nano的帮助文档中，Ctrl键使用”^”标识，Alt键使用”M”标识（Meta）。

nano 命令语句

  执行 nano 命令，可以加入很多选项，上文中提到的 -w 便是其中一个，如果想知道 nano 命令都支持哪些选项，可以使用 nano -h 命令来查看

打开、保存、关闭文件

  • 打开 ：如上文所说，打开文件使用 nano -w filename

  • 保存 ：使用快捷键 Ctrl+O

  • 退出 ：使用快捷键 Ctrl+X

  如果你修改了文件，下面会询问你是否需要保存修改，输入 Y 确认保存，输入 N 则不保存，按 Ctrl+C 取消并返回。
  如果输入了 Y 保存，下一步会让你输入保存的文件名，若不需要修改文件名直接回车即可；若要保存新文件或重命名（另存）则输入确认即可。此时 Ctrl+C 同样可以取消并返回。

移动光标

  使用方向键移动。

编辑文本

  在光标处直接敲击键盘输入即可，会在光标所在字符前增加输入的字符。

搜索

  使用快捷键 Crtl+W 输入你要搜索的关键字，回车确认。会定位到第一个匹配的文本，可使用 Alt+W 来定位下一个匹配的文本。

复制、剪切、粘贴

  • 复制 ：Alt+6 （单独使用复制一行）

  • 剪切 ：Ctrl+K （单独使用剪切一行）

  • 粘贴 ：Ctrl+U

复制、剪切某部分

  方向键将光标移动到文本开头，使用 Alt+A 标记开头，使用方向键移动光标到文本末尾，此时选中的文本会反白，使用 Alt+6 复制 或 Ctrl+K 剪切。使用 Ctrl+U 来粘贴。
  Alt+A 一次开始标记，再执行Alt+A 取消标记。

想了解更多操作，可使用快捷键 Ctlr+G 查看其它功能。

vim

打开、保存、关闭 文件

  • vi filename 打开filename文件
  • vi +n filename 打开文件，并将光标置于第n行首
  • vi + filename 打开文件，并将光标置于最后一行首

  • :w 保存文件
  • :w filename 保存至filename文件

  • :n1,n2 w filename 将n1到n2行的内容另存为filename文件

  • :q 退出编辑器
  • :q! 强制退出编辑器，且不保存
  • :wq 退出编辑器，并保存文件

移动光标

  • h、j、k、l 左、下、上、右

  • zz 把光标下的文字置于屏幕中央

  • Ctrl+b 屏幕往后移动一页。

  • Ctrl+f 屏幕往前移动一页。

插入文本或行

  • a 在当前光标位置的右侧添加文本

  • i 在当前光标位置的左侧添加文本

  • A 在当前行的末尾位置添加文本

  • I 在当前行的开始处添加文本（非空字符的行首)

  • O 在当前行的上面新建一行

  • o 在当前行的下面新建一行

删除、恢复字符或行

  • x 删除当前字符

  • dd 删除当前行

  • u 撤销上一步操作

搜索

  • /.* 向光标下搜索

  • ?.* 向光标上搜索

  • n 向下搜索前一个搜索动作

  • N向上搜索前一个搜索动作

替换

  • :s/old/new 用new替换行中首次出现的old

  • :s/old/new/g 用new替换行中所有的old

  • :n,m s/old/new/g 用new替换从n到m行里所有的old

  • :%s/old/new/g 用new替换当前文件里所有的old

  • R 替换、覆盖当前光标位置及后面的若干文本

  • r 取代光标所在处的单个字符

跳转到制定行

  • gg 跳到第一行

  • nG 跳到行号为n的行

  • G 跳转至文件底部

复制、粘贴

  • yy 复制当前行
  • p 粘贴剪切板里的内容在光标后

vi 选项

  • :set nu 显示行号
  • :set nonu 不显示行
