Win10 中使用 Hyper-V 创建 Linux 虚拟机

发表于 2019-05-17 | 分类于 绝

简介

Hyper-V 是微软的一款虚拟化产品，是微软第一个采用类似 Vmware 和 Citrix 开源 Xen 一样的基于 hypervisor 的技术。这也意味着微软会更加直接地与市场先行者 VMware 展开竞争，但竞争的方式会有所不同。

Hyper-V 是微软提出的一种系统管理程序虚拟化技术，能够实现桌面虚拟化。Hyper-V 最初预定在2008年第一季度，与 Windows Server 2008 同时发布。Hyper-V Server 2012 完成 RTM 版发布。

本帖使用 Windows10 系统中的 Hyper-V 进行虚拟机的搭建。Windows 7 用户同样也可以使用 Hyper-V 进行虚拟机的搭建，但需要去微软官网另行下载 远程服务器管理工具的更新补丁包。Win7 的使用过程可参考此帖。

环境

  • Windows 10 专业版

准备

CentOS 7 的 iso 文件

CPU 虚拟化支持

  • 使用 Hyper-V 需要 CPU 支持虚拟化，并开启此功能，可以在任务管理器中查看是否开启了虚拟化。

  • Hyper-V 与 VMware 是不能同时使用的，不过网上也有两者互容的解决方案。

开启 Hyper-V 功能

  • 进入 控制面板 -> 程序和功能 -> 启用或关闭 Windows 功能
  • 将 Hpyer-V 复原框选中，启用 Windows 的 Hyper-V 功能

步骤

创建虚拟机

  • 启动 Hyper-V 管理器

    在【开始】中搜索“Hyper-V”即可

  • 进入 Hyper-V 管理器界面

  • 创建虚拟机

    操作系统运行在硬件设备上，下面通过 Hyper-V 想到来创建一个虚拟的硬件设备。
    点击右侧操作区域中的“新建”/“虚拟机”

  • 按照“新建虚拟机向导”一步步执行

  • 填写虚拟机名称与虚拟机的存储位置

  • 指定一个虚拟机的启动内存。我设置成了2048MB。勾选动态内存，节约硬件资源。

  • 选择“未链接”（联网的设置方法在其他章节介绍）

  • 找到事先准备好的 CentOS7 的 iso 系统镜像文件

  • 点击“完成”后开始创建虚拟机

  • 完成后在 Hyper-V 的界面中间部分“虚拟机”中会有你刚才命名的虚拟机。

安装 CentOS 系统

  • 在“创建虚拟机”中我们已经将 CentOS7 的 iso 镜像放入了虚拟的光驱中。

  • 选中创建好的虚拟机，点击右侧“启动”按钮。

  • 然后点击上面的 “链接…”按钮，弹出一个虚拟机的视图窗口

  • CentOS 7 的具体安装过程可以参考这个文章
    注意安装软件包的时候不建议选择最小化和图形化两种安装方式，可以选择基础web服务

  • 安装完成后，会出现登录的界面

创建网络

新建虚拟网络交换机

  • 点击右侧“操作”选项中“虚拟交换机管理器…”

  • 点击“新建虚拟网络交换机”，选择“外部”并“创建虚拟交换机”

  • 填写一个名称，勾选“链接类型”中的“外部网络“，之后点击确定

  • 给虚拟机选取网络
